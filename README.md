# BioPetNutrition

Одностраничный сайт о питании животных и биохимии, созданный для предоставления научно обоснованной информации о рационе и здоровье домашних и сельскохозяйственных животных.

## Особенности

- Тёплый коричневый профессиональный дизайн.
- Баннер с тематическим изображением (собаки и кошки).
- Секции:
  - Биохимия питания животных
  - Ветеринарные исследования
  - Практические рекомендации
- Галерея с фотографиями собак и кошек.
- Контактная форма, работающая через [Formspree](https://formspree.io/).
- Плавная навигация по сайту.

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/username/BioPetNutrition.git

2. Откройте index.html в браузере.
 3. Для работы формы:
 • Зарегистрируйтесь на Formspree.io (https://formspree.io/)
 • Создайте форму и получите ID
 • Замените yourformid в index.html на ваш Formspree ID

Стек технологий
 • HTML5
 • CSS3
 • JavaScript
 • Formspree (для формы)
