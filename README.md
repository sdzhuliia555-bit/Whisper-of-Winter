# ❄️ Шёпот зимы

Атмосферный одностраничный сайт, вдохновлённый зимой — её тишиной, теплом и магией природы.

## 🌿 Что внутри
- Мягкая цветовая палитра зимнего дня  
- Галерея животных зимой (олень, лиса, сова)  
- Анимация падающего снега  
- Плавные движения изображений при скролле  

## 🛠️ Технологии
- HTML5  
- CSS3  
- JavaScript
